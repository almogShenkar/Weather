Welcome WeatherLib

WeatherLibary is a c# library that provides your app weather service from all over the world. By entering the request location you can get the weather right away.

Easy to install:

Just Download and add reference from your original project. log4net package is required(Nuget packages).

Simple to use:

var location = new Location(Console.ReadLine());
OpenWeatherMapService.SetKey("key from the web service- for example openweathermap");
var service =
WeatherDataServiceFactory.GetWeatherDataService(WeatherDataServiceFactory.OPEN_WEATHER_MAP); var data = service.GetWeatherData(location);

Log file: Weather/sample/bin/debug/OpenWeatherMapService.txt
